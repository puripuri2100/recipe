# recipe

## 概要

このリポジトリには、「各家や人にはそれぞれに適した素材選びと味付けがあり、それぞれにレシピサイトがあるべきだ」という考えのもと作成されたレシピサイト構築用のコードと、作者のために調整されたレシピの情報が含まれています。

今登録されているレシピはこちらから見ることができます。

- <https://puripuri2100.github.io/recipe/>

## 自分だけのレシピサイトを立ち上げる

以下の手順により、自分だけのレシピサイトを立ち上げることができます。

1. "Fork"ボタンをクリックし、リポジトリを複製する
2. "Settings"ボタンをクリックし、設定ページに飛ぶ
3. "Pages"ボタンをクリックし、GitHub Pagesを有効化する
4. "Build and deployment"の中の"Source"を"GitHub Actions"に設定する
5. `src/recipes`内にレシピ情報を書いたMarkdownファイルを置く

## レシピを提案する

新しいレシピを提案する場合は[Issues](https://github.com/puripuri2100/recipe/issues)に提案してみてください。
作ってみて良さそうなときには登録します。
